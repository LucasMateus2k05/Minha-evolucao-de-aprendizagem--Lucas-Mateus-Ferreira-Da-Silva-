## Minha evolução: o que aprendi sobre os assuntos que foram abordados nesse módulo?

- FUNÇÕES:
    Aprendi que funções são blocos de código que podem ser reutilizados em diferentes partes do programa, Também Entendi que funções podem ter alguns parâmetros e que esses parâmetros podem ser utilizados dentro da função, foi abordado durante as aulas sobre como que fazemos para criar e chamar funções com retorno.
    Utilizamos as funcoes para resolvermos problemas especificos e para deixarmos o codigo mais suguro utilizamos as tipagens nas funcoes.


    - TIPOS DE DADOS:
    Aprendi que existem diferentes tipos de dados em JavaScript/typescript como os tipos literias, onde tambem podemos fazer unioes dos tipos.
    aprendi sobre as tuplas que são estruturas de dados que permitem combinar diferentes tipos de elementos em um tipo específico e com tamanho fixo.
    Aprendi tambem sobre os tipos conjuntos, Narrowing, Assertione Unknown.


    - GESTÃO DE TEMPO:
    Aprendi como criar um cronograma e como priorizar tarefas.
    Aprendi como utilizar ferramentas de gestão de tempo, como o metodo Pomodoro,


    - TIPOS UTILITARIOS:
    Aprendi como utilizar os tipos utilitarios em TypeScript e que ao utilizar os tipos utilitarios podemos melhorar a segurança e a legibilidade do código.
    alguns dos tipos apresentados foram os :
    - Partial -->  Resulta em um novo tipo manipulado;
    - Required -->  Faz com oque o tipo tenha todas as propriedades como obrigatório;
    - Readonly -->  nao da para mudar pois o tipo utilitario "Readonly" permite apenas a leitura dos dados e não permite a alteração, Serve como uma segurança a mais para o codigo;
    - Pick -->  Ele adiciona as propriedades de um tipo á um novo tipo criado;
    - Omit -->  Ele é utilizado para criar um novo tipo que é uma versão modificada de outro tipo, com propriedades selecionadas para serem excluídas;
    - Record -->  É utilizado para que possamos criar tipos de objetos de forma dinâmica em TypeScript;
    - Exclude -->  É utilizado em TypeScript para excluir itens específicos de uma união de tipos;
    - Extract -->  Este é utilizado para extrair itens específicos de uma união de tipos, criando assim um novo tipo apenas com os itens extraídos.


    - METODOS STRINGS:
    Aprendi como utilizar os métodos de strings em JavaScript e no TypeScript, como os métodos
    toUpperCase(), Método substring(), Método trim(), Método trimStart() e trimEnd(), Método toUpperCase e toLowerCase, Método substring, Método Slice, Método split, Método Replace e ReplaceAll, Método padStart e padEnd, Método includes e indexOf.
    Aprendi como utilizar os métodos de strings para manipular texto na programação.


    - CURRICULO E CARTA DE APRESENTAÇÃO:
    Aprendi como criar um currículo e uma carta de apresentação e a como me destacar minhas habilidades e experiências em um currículo.

    - METODOS ARRAY I. II e III.
    No metodo array I, aprendi como utilizar os métodos de array em JavaScript e TypeScript, como os métodos:
    push, pop, shift, unshift, indexOf, includes, reverse, join, slice, splice.

    no metodo array II, aprendi um pouco sobre os conceitos de callbacks com setTimeout e setInterval e alguns metodos como:
    Every, Some, FindIndex, Filter e Map.

    No metodo array III, aprendi sobre ordenação padrao com o metodo sort, sobre as ordenaçoes numericas e tambem por subtração, aprendi tambem como funciona o reduce na pratica e em uma matriz,aprendi sobre como ordenar uma string em forma crescente e decrescente.

    Aprendi como utilizar os métodos de array para manipular arrays em
    programação.
